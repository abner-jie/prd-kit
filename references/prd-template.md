# 跨团队 PRD 模板

用于 CEO、运营、品牌、市场、合作团队、客服、开发和 QA 都需要阅读的复杂 PRD。

## 目录

- 元信息
- 1. 一页摘要
- 2. 问题定义
- 3. 用户和场景
- 4. 需求范围
- 5. 产品规则
- 6. 验收标准
- 7. 风险和停止条件
- 8. 待确认问题
- 9. 变更记录
